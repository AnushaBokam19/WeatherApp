This project is a Vite + React prototype for a glassmorphic SaaS weather product powered by Weatherstack.

Quick start:
- npm install
- npm run dev

Features:
- Current weather, Historical, Forecast, Marine, Locations endpoints (Weatherstack)
- Search, filters, basic autocomplete
- Glassmorphic UI (neutral, realistic SaaS look)

API key is read from VITE_WEATHERSTACK_KEY in `.env`.

Docs: https://docs.apilayer.com/weatherstack/docs/api-documentation
